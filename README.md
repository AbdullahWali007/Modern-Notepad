**Modern Notepad** is a lightweight yet feature-rich desktop text editor designed with a focus on modern UI/UX principles and modular software architecture. Built using **Python**, **CustomTkinter**, and **Tkinter**, it provides a high-performance environment for rapid note-taking and code editing.

The application leverages a composite UI design pattern to manage a dynamic tabbed interface, allowing users to handle multiple files simultaneously within a single instance. By integrating the **pywinstyles** library, the editor supports native Windows 11 Mica and Acrylic transparency effects, delivering a sleek, "Apple-esque" aesthetic that aligns with contemporary design standards.

### Key Features

* **Multi-Tab Interface**: Manage numerous text files efficiently using a seamless tabbed navigation system.
* **Real-Time Line Numbering**: Integrated canvas-based line tracking that updates dynamically with text input and scrolling.
* **Aesthetic Customization**: Comprehensive settings panel to adjust font families, font sizes, and toggle between high-contrast Dark and Light modes.
* **Dynamic Find and Replace**: A dedicated utility for searching and batch-replacing strings across active document buffers.
* **Integrated Tools**: Includes a built-in Typing Speed Check utility to measure Words Per Minute (WPM) and accuracy in real-time.
* **Cross-Platform Compatibility**: Optimized keyboard shortcuts and UI scaling for Windows, macOS, and Linux environments.
* **Safe File Handling**: Robust file I/O operations with UTF-8 encoding support to ensure data integrity across various character sets.

### Technical Stack

* **Language**: Python
* **GUI Framework**: CustomTkinter / Tkinter
* **Styling**: PyWinStyles (Mica/Acrylic support)
* **Architecture**: Object-Oriented (Composite Pattern)
